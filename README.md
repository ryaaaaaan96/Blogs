## 学习路径建议

### 嵌入式开发入门
```
学习路线(C语言→数据结构) → 裸机调试 → RTOS → STM32
```

### ESP32 物联网开发
```
ESP32环境搭建 → WiFi → 蓝牙 → 传感器 → 多任务
```

### 网络协议栈
```
LwIP基础 → 内存管理 → pbuf → 网络接口 → 协议核心
```

### 编程规范提升
```
MISRA C → 事件驱动编程
```
